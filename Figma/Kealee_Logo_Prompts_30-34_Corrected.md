## PROMPT 33: Logo — Animated Version

Create a subtle animated version of the Kealee logo for digital use.

**BASE LOGO:**
- Bubble icon cluster (orange, teal, green with navy outlines)
- "Kealee" wordmark in bubbly rounded font
- "CONSTRUCTION PLATFORM" tagline
- Bubbly, friendly aesthetic

**ANIMATION CONCEPT — "Building Bubbles":**
The bubbles appear to "build up" or "bubble up" to form the logo, using the platform's warm-to-cool color progression.

**ANIMATION SEQUENCE (2–3 seconds total):**

| Time | Action | Details |
|------|--------|---------|
| 0.0–0.3s | Small green accent dot appears | Slight bounce, elastic ease |
| 0.3–0.6s | Green bubble (#38A169) scales up from dot | Elastic overshoot |
| 0.6–0.9s | Teal bubble (#2ABFBF) slides in, overlaps | Smooth ease-out |
| 0.9–1.2s | Large orange bubble (#E8793A) scales up behind | Fills dominant position |
| 1.2–1.5s | Navy outlines (#1A2B4A) draw on around each bubble | Stroke animation, clockwise |
| 1.5–2.0s | "Kealee" wordmark types in or fades letter by letter | Bubbly font, left to right |
| 2.0–2.5s | "CONSTRUCTION PLATFORM" tagline fades in | Subtle, all at once |
| 2.5–3.0s | Subtle pulse/glow on complete logo → static | Single pulse, then done |

**ANIMATION PRINCIPLES:**
- Elastic/bouncy easing (matches bubbly brand)
- Each bubble has slight overshoot on scale (like a soap bubble appearing)
- Outlines draw on smoothly with consistent speed
- Color order: green → teal → orange (cool to warm, small to large)
- Final state is perfectly static (no endless loops)
- Plays ONCE on load — does not repeat

**HOVER STATE (for web navigation):**
- Subtle float animation on bubbles (gentle up/down, 2px oscillation)
- Slight brightness increase on hover (+5%)
- Smooth transition in/out (0.3s ease)

**WHERE THIS PLAYS ON THE PLATFORM:**

- **Navbar logo:** Static with hover float state only
- **Footer logo:** Static, no animation

**APP ICON REQUIREMENTS:**

**iOS App Icon:**
- 1024x1024px master
- No transparency (Apple requirement)
- Rounded corners applied by OS
- Ensure icon is centered and fills space well
- Background: White (#FFFFFF) or subtle warm white gradient

**Android Adaptive Icon:**
- 512x512px with safe zone consideration (66% inner safe area)
- Foreground: Bubble icon cluster
- Background: White or subtle radial gradient (white → warm gray)
- Account for circular, squircle, and rounded square masks

**Favicon Package:**
- 16x16px — Simplified (see simplification rules below)
- 32x32px — Simplified
- 48x48px — Full icon
- 64x64px — Full icon
- 180x180px — Apple touch icon

- 512x512px — PWA icon

**SIMPLIFICATION RULES FOR SMALL SIZES:**

| Size | Approach |
|------|----------|
| 16px | Single orange circle with subtle teal accent arc — recognizable as "K bubble" |
| 32px | Two bubbles (orange + teal) with navy outline — no green, no dots |
| 48px+ | Full icon with all three bubbles, outlines, and accent dots |

Key rules:
- Remove smallest elements first (dots, then green bubble)
- Thicken outlines proportionally (+1px per reduction step)
- Maintain orange as dominant color at every size
- Must be recognizable as Kealee at every size

**BACKGROUND OPTIONS:**
1. White (#FFFFFF) — primary, clean
