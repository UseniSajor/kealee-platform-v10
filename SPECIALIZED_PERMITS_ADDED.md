## 📋 New Permit Types Added

### 1. **Zoning & Variance Submissions** - $750
Coordinate zoning variance requests and special use permits
- Variance application preparation
- Zoning board presentation support
- Public hearing coordination
- Neighbor notification requirements
- Approval tracking

### 2. **Historic Review Submissions** - $650
Historic preservation review board submissions for properties in historic districts
- Historic review board applications
- Secretary of Interior Standards compliance
- Design guideline verification
- Board presentation coordination
- Certificate of Appropriateness processing

### 3. **Subdivision Submissions** - $1,200
Comprehensive subdivision plat review and approval coordination
- Preliminary plat submission
- Final plat submission
- Engineering review coordination
- Utility approval coordination
- Planning commission presentation

### 4. **Land Use Approvals** - $850
Land use and site plan review approvals
- Site plan review applications
- Conditional use permits
- Special exception requests
- Planning department coordination
- Approval condition tracking
