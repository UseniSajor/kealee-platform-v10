## 🎯 THE MARKETPLACE QUESTION

### **What is m-marketplace?**

**m-marketplace is the FUTURE main offering - a two-sided marketplace connecting:**
- **Supply Side:** Contractors, architects, engineers, specialty trades
- **Demand Side:** Homeowners, investors, developers

**Current State (Launch):**
- m-ops-services = Main offering (Package A/B/C/D with Kealee PMs)
- m-marketplace = Feature within platform (hire vetted contractors)

**Future State (Year 2+):**
- m-marketplace = Primary growth engine
- m-ops-services = Premium tier (with dedicated PMs)
