## 🔧 TROUBLESHOOTING GUIDE

### Common Issues

**Issue: Can't upload large files**
- Solution: Files should be < 100MB. Compress or split larger files.

**Issue: Version comparison is slow**
- Solution: Limit comparison to specific files, not entire project.

**Issue: Template not appearing**
- Solution: Check template status is PUBLISHED or isStandard is true.

**Issue: Permit package submission fails**
- Solution: Verify all application forms are complete and verified.

**Issue: Stamp application fails**
- Solution: Ensure license is validated and seal is uploaded.

## 📊 PERFORMANCE TIPS

### For Large Projects
- Use version branches for experimental work
- Limit concurrent file operations
- Use random sampling for QC on large sets
- Archive old versions regularly

### For Teams
- Assign clear roles and permissions
- Use comments for communication
- Document all design decisions
- Keep meeting minutes updated

### For File Management
- Organize files by discipline
- Use consistent naming
- Keep file sizes reasonable
- Archive completed projects
