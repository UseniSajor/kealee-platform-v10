## Overview

Three products (Design Concept, Estimation, Permits) each follow the same 3-tier structure.

## DESIGN CONCEPT Tier 1 — Basic ($99–$299)

**Delivery:** 3–5 business days | **Target:** Budget-conscious explorers

### Deliverables

1. **Professional PDF Report (15–20 pages)**
   - Cover & Executive Summary
   - Project Vision Statement (style, color palette, key features)
   - Spatial Planning (dimensions, layout, work zones, storage)
   - Zoning & Compliance (jurisdiction, setbacks, code compliance)
   - Materials & Finishes (cabinetry, countertops, flooring, appliances, lighting)
   - Budget Summary (category breakdown, regional labor adjustment)
   - MEP Systems Overview (electrical circuits, plumbing, HVAC)
   - Timeline & Phases (5 phases, 12–16 weeks typical)
   - Project Assumptions & Limitations
   - Next Steps & Support Info
   - Appendix (glossary, material brands)

2. **Conceptual Renderings (3–5 images, 1920×1080px)**
   - Front counter view
   - Island & seating view
   - Full kitchen overview
   - Detail shot (optional)
   - Alternate angle (optional)

3. **Quick Reference Sheet (1 page)**
   - At-a-glance: style, cost, timeline, key materials, permits

4. **Budget Comparison Chart**
   - Budget version (save ~$8K)
   - Standard version (current estimate)
   - Luxury version (add ~$12K)

5. **Design Rationale Notes (1–2 pages)**
   - Why this design, materials, and color scheme

6. **Web Portal Access**
   - Shareable link, lifetime cloud storage, download links

**Inclusions:** 1 revision, email support, 10-year cloud access

## DESIGN CONCEPT Tier 2 — Premium ($349–$899) ⭐ MOST POPULAR

**Delivery:** 3–5 business days | **Target:** Most homeowners (80% choose this)

**Everything in Tier 1, PLUS:**

### Additional Deliverables

7. **60-Second Transformation Video (1080p MP4)**
   - Segment 1 (0–5s): Kealee intro, "Your [Kitchen] Reimagined"
   - Segment 2 (5–15s): Before tour with 360° angles
   - Segment 3 (15–50s): HeyGen avatar narration (30s) + Runway ML transformation (20s)
   - Segment 4 (50–60s): Final reveal, branding, CTA
   - Professional color grading, licensed music, subtitles
   - Download + streaming + social-ready formats

8. **2D Architectural Floor Plan (SVG + PDF, scale 1/4" = 1')**
   - Structural elements (walls, doors, windows)
   - Room labels with dimensions
   - Cabinetry layout (base, wall, island)
   - Countertops & backsplash
   - Appliances (symbols + labels)
   - MEP systems color-coded:
     - 🟠 Electrical: outlets (15+), switches, fixtures, circuits
     - 🔵 Plumbing: supply lines, drains, fixtures, shut-offs
     - 🟢 HVAC: hood, ducting, supply/return vents
   - Full dimensions, legend, north arrow, scale bar
   - Interactive viewer (zoom/pan) + PDF download

9. **Enhanced Renderings (6–8 images, 2560×1440px)**
   - Hero shot, island detail, appliance zone, sink area, seating, full context, alternate lighting, specialty view
   - Social/print/mobile download formats

10. **Interactive Web Portal (kealee.com/concepts/[ID])**
    - Embedded video player, interactive floor plan with layer toggles
    - Renderings gallery (lightbox), specifications tabs (8 tabs)
    - All document downloads, sharing features, lifetime access
    - Mobile-friendly, password protection option

11. **Detailed Bill of Materials (8 sections, Excel + PDF)**
    - Cabinetry, Countertops, Appliances, Sinks & Faucets, Flooring, Lighting & Electrical, Plumbing, Miscellaneous
    - Unit prices, quantities, labor, totals per item

12. **Material Comparison & Upgrade Tool**
    - Interactive budget/standard/luxury comparison with cost impact

**Inclusions:** 1 major + 2 minor revisions, email + phone support (30 days), 10-year cloud

## Platform Delivery Capability

| Component | Method | COGS |
|---|---|---|
| Reports | DesignBot (Claude Opus) → HTML → PDF | $10–15 |
| Renderings | AI image generation (Midjourney/SD) | $5–10 |
| Video | HeyGen API + Runway ML batch | $5–10 |
| Floor plan | SVG from DesignBot JSON output | $0 |
| Portal | CMS template (auto-creates) | $0.10 |
| Support | FAQ bot handles 80% | $20–40 |

**Margin:** 400–2000% depending on tier. Fully automated. Scales to 10K+ projects with negligible incremental cost.

4. Budget (cost breakdown)
5. Timeline (project phases)
6. MEP Systems (electrical, plumbing, HVAC)
7. Zoning (compliance verification)
8. Code (applicable building codes)
